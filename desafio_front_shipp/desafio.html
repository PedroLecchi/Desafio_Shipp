Piscina de Coisas Comportamentais

Na piscina abaixo, deverão ser criados nós iguais ao template .thing já existente abaixo sempre que o botão "Adicionar" for clicado. Essa nova coisa deve vir com uma cor de fundo aleatória, ao invés do cinza padrão do template.

O botão remover todos limpa a piscina.

Quando um nó for adicionado, o <body> deve informar na <div id="info"> a mensagem: "A piscina contém X coisas.", onde X é o número de nós que a psicina tem.

Caso os nós sejam removidos, o <body> não deve atualizar a info.

As coisas da piscina tem um botão "color" que ao ser clicado deve escolher uma cor aleatória para o elemento.
E o link remover apaga a coisa específica da piscina.

Tudo isso deve ser implementado com apenas 3 "Listeners", sem o auxílio de qualquer framework externo.

Esse html não pode ser editado, apenas programaticamente.
